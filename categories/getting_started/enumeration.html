როგორ დავიწყოთ

რა გვჭირდება CTF დავალებების გასაკეთებლად

  1. არტიკლები
  2. როგორ დავიწყოთ

ხელსაწყოები და მათი გამოყენება

CTF დავალებების მიხედვით ბევრი სხვადასხვა ხელსაწყოს გამოყენება შეიძლება მოგიწიოთ, მათ შორის სკანირება, კრიპტოგრაფია, ექსპლუატაცია და კომპიუტერული ექსპერტიზა (Forensics). ამისდა მიუხედავად ბევრი CTF დავალების საწყისი მიდგომა მსგავსიაა. აქ ჩამოვთვლით ყველაზე ხშირად გამოყენებად ხელსაწყოებს და მათ გამოყენებას:

  • nmap: ქსელის სკანერი, რომლითაც ვამოწმებთ რომელი პორტებია ღია და რა სერვისები გვხვდება.
  • Burp Suite: საუკეთესოა Web-CTF-ისთვის. გვაძლევს შესაძლებლობას ვმართოთ მოთხოვნები, ვნახოთ რა იგზავნება და მივიღოთ შესაბამისი პასუხი.
  • CyberChef: ონლაინ ინსტრუმენტი მონაცემების დეკოდირებისა და დაშიფვრისთვის. მაგალითად base64, rot13, hex, jwt და ა.შ.
  • ghidra: რევერსინგისთვის გამოსადეგი ინსტრუმენტი. ვნახოთ, როგორ მუშაობს პროგრამა და როგორ ავუღოთ თავები compiled ბინარებზე.
  • strings, file, ltrace, strace: ესენი ძირითად unix ხელსაწყოებია, რაც საშუალებას გვაძლევს ვნახოთ რა ფუნქციებს იძახს პროგრამა და გავიგოთ მეტი მის შესახებ.
  • pwntools: Python ბიბლიოთეკა ექსპლუატაციისთვის. მრავალ ფუნქციას აერთიანებს და ხშირად გამოიყენება pwn კატეგორიაში.
  • binwalk: გამოსადეგია სურათებში ან ბინარულ ფაილებში ინფორმაციის მოსაძებნად.
  • steghide, zsteg, exiftool: გამოსადეგია stego კატეგორიისთვის, როდესაც ინფორმაციას მალავენ სურათში ან ფაილის მეტამონაცემებში.

ამ ხელსაწყოების გარდა მნიშვნელოვანია გქონდეთ Linux-ის საბაზისო ცოდნა და Bash-ის გამოყენების გამოცდილება, რადგან ხშირად დაგჭირდებათ სკრიპტების წერა, ფაილების ანალიზი ან სხვადასხვა command chaining.
