როგორ დავიწყოთ CTF ების გაკეთება ინდივიდუალურად?

ოპერაციული სისტემა და "ლაბორატორიის" მოწყობა

კიბერუსაფრთხოების სხვადასხვა საკითხებისთვის თითქმის ნებისმიერი ტიპის ოპერაციული სისტემის გამოყენებაა შესაძლებელი, თუმცა საზოგადოების სიყვარულს დღემდე იმსახურებს Linux ზე დაფუძვნებული ოპერაციული სისტემები.

Linux პოპულარულია რადგან:

  • აქვს ე.წ ღია კოდი (Open Source) რაც იმას ნიშნავს რომ სისტემაზე მომხმარებელს აქვს სრული კონტროლი და აქვს საშუალება რომ შეცვალოს სისტემის ნებისმიერი კომპონენტი თავის სასარგებლოდ.
  • აქვს ჩაშენებული უსაფრთხოების ფუნქციები მაგალითად ძლიერი მომხმარებლის დაშვების სისტემა (User permission system) პროცესების იზოლაცია და ნაკლები პროგრამები უკანა ფონზე.
  • აქვს ჩაშენებული ტერმინალი
  • სერვერების და ქსელების უმეტესობა არის კონფიგურირებული ლინუქსზე შესაბამისად ლინუქსის ცოდნა აუცილებელია მსგავსი ტიპის მოწყობილობების შეღწევადობის ტესტირებისთვის.

რეკომენდირებული Linux დისტრიბუციები

Kali Linux

საუკეთესოა დამწყებებისთვისაც და კიბერუსაფრთხოების ექსპერტებისთვისაც. ასევე საუკეთესოა შეღწევადობის ტესტირებისთვის.

Parrot OS

შედარებით მსუბუქი ალტერნატივა Kali Linux თან უფრო მსუბუქია რაც იმას ნიშნავს რომ ნებისმიერი ტიპის კომპიუტერზე ლეპტოპზე და ტელეფონზე შეიძლება დაყენება თუმცა აქვს უფრო ნაკლები "ხელსაწყოები" ვიდრე Kali Linux ს

Ubuntu ან სხვა

მიუხედავად იმისა რომ ისეთი ოპერაციული სისტემებს როგორიც არის Kali ან Parrot OS ჩაწერის მომენტიდან მოყვება ძალიან ბევრი ხელსაწყო, მსგავსი გარემოს მოწყობა ნებისმიერი ტიპის ლინუქსის დისტრიბუციაზე არის შესაძლებელი.

რეკომენდირებული აპლიკაციის ტიპები

დეკომპილატორები/დამშლელები

აპლიკაციები რომლებიც მანქანურ კოდს შლის და შეძლებისდაგვარად "თარგმინს" ადამიანურ კოდის ენაზე. მსგავსი აპლიკაციის მაგალითია Binaryninja

დებაგერები

Debugger ანუ დებაგერი არის აპლიკაცია რომელიც გვაძლევს საშუალებას ვნახოთ რა ხდება აპლიკაციის "შიგნით". მსგავსი მაგალითია GDB

ვებ აპლიკაციის ანალიზი

ამ კატეგორიაში ბევრი სხვადასხვა ტიპის ხელსაწყო შედის მაგალითად Nikto და Burp Suite

ქსელის ანალიზი

არსებობს აპლიკაციები რომლებსაც შეუძლიათ ქსელის ნაკადის "დაჭერა" და ანალიზი. ამ მიზნისთვის საუკეთესო აპლიკაციაა Wireshark
